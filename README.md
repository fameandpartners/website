## Welcome to Fame and Partners

## Getting Started

### Setup project
```
$ git clone git@github.com:droidlabs/fame_and_partners.git
$ cd ./fame_and_partners
$ cp config/database.yml.example config/database.yml
$ bundle install
```

### Setup database
```
$ rake db:setup
$ rake spree_sample:load
```
